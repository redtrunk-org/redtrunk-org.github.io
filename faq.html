About Browse View Download FAQ

The About, Browse, View, Download, and FAQ menu tabs can be selected with keyboard shortcuts. The View page image navigation and zoom-in/out can also be keyboard controlled.

A shortcut combines a unique site access key with a browser-specific key combination to initiate the action.

Access Keys

Menu Shortcuts View Page Shortcuts
a About [ previous image
b Browse ] next image
v View s zoom full size
d Download w zoom full width
q FAQ o zoom out
f download file
